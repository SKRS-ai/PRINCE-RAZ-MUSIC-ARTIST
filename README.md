## 🏛 The Vision
PRINCE RAZ represents the intersection of luxury and sound. Built during the M3-Hackathon-2021, this portal serves as a 24/7 "living" press kit that eliminates traditional site navigation in favor of an immersive, gold-accented broadcast experience.

## 👑 Features
- **Cinematic Loop Engine:** Seamless high-definition video and audio cycling with zero latency.
- **Royal Design Language:** A premium gold-on-black interface using elegant typography (Cinzel & Montserrat).
- **The Palace Hub:** Direct access to professional media assets, booking protocols, and high-res press photos.
